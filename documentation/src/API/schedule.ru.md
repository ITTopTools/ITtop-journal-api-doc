# Расписание

Расписание занятий по дате и диапазону дат.

## Эндпоинты

### GET /schedule/operations/get-by-date

Расписание занятий на определённую дату.

**Назначение:** Просмотр расписания на день — отображает занятия с указанием времени, преподавателя, предмета и номера аудитории.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `date_filter` | string | Дата в формате `YYYY-MM-DD` |

**Ответ:** Массив объектов с полями `date`, `lesson` (номер занятия), `started_at`, `finished_at`, `teacher_name`, `subject_name`, `room_name`.

### GET /schedule/operations/get-by-date-range

Расписание занятий на диапазон дат.

**Назначение:** Просмотр расписания на неделю/месяц — возвращает занятия между начальной и конечной датами включительно. Возвращает пустой массив, если на указанный диапазон нет занятий.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `date_start` | string | Начальная дата в формате `YYYY-MM-DD` |
| `date_end` | string | Конечная дата в формате `YYYY-MM-DD` |

**Ответ:** Массив объектов занятий (та же структура, что и у get-by-date). Пустой массив является нормальным результатом для диапазонов дат без занятий.

### GET /schedule/operations/get-month

Расписание занятий на месяц, содержащий указанную дату.

**Назначение:** Обзор расписания на месяц — та же структура элементов, что и у get-by-date.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `date_filter` | string | Любая дата в формате `YYYY-MM-DD` в пределах целевого месяца |

**Ответ:** Массив объектов занятий (та же структура, что и у get-by-date).
