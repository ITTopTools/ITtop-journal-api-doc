# Публичные данные

Публичные справочные данные — аутентификация не требуется.

## Эндпоинты

### GET /public/languages

Доступные языки интерфейса.

**Назначение:** Справочник для выбора языка — список локалей, поддерживаемых приложением.

**Ответ:** Массив объектов с полями `name_mystat` (отображаемое имя), `short_name` (код локали, например "ru").

### GET /public/translations

Пары ключ-значение для локализации на заданном языке.

**Назначение:** Строки переводов интерфейса. Возвращает плоский словарь ключей перевода и соответствующих переведённых строк.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `language` | string | Код локали (например "ru"). Собраны данные только для русского; другие локали имеют такую же схему. |

**Ответ:** Плоский словарь строковых ключей и строковых значений (карта переводов).

### GET /public/tags

Справочный список публичных тегов.

**Назначение:** Справочные данные тегов, используемые в приложении.

**Ответ:** Массив элементов тегов.
