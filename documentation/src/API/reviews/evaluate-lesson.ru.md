# GET /feedback/students/evaluate-lesson-list

Получить список уроков, доступных для оценки учеником.

## Назначение

Используется формой обратной связи для отображения прошедших уроков, где ученик может оценить преподавателя. Каждая запись содержит дату урока, имя преподавателя и предмет.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `key` | string | Уникальный идентификатор урока |
| `date_visit` | string | Дата проведения урока |
| `fio_teach` | string | Полное имя преподавателя |
| `spec_name` | string | Название предмета |
| `teach_photo` | string | URL фото преподавателя |

**Тип ответа:** Массив объектов
