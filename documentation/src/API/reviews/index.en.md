# Reviews

Lesson reviews and student feedback systems — view teacher reviews, submit feedback, and track social review submissions.

## Endpoints

| Method | Path | Summary | Link |
|--------|------|---------|------|
| GET | `/reviews/index/list` | List lesson reviews | [Details](list.md) |
| GET | `/reviews/index/instruction` | Get review guidelines | [Details](instruction.md) |
| GET | `/feedback/students/evaluate-lesson-list` | List lessons available for evaluation | [Details](evaluate-lesson.md) |
| GET | `/feedback/social-review/get-review-list` | List social review submissions | [Details](social-review.md) |
