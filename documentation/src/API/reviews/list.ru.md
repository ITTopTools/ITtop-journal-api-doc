# GET /reviews/index/list

Получить список отзывов об уроках ученика.

## Назначение

Отображает отзывы преподавателей на странице профиля ученика. Каждая запись содержит дату отзыва, текст сообщения, предмет и имя преподавателя.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `date` | string | Дата отзыва |
| `message` | string | Текст отзыва |
| `spec` | string | Краткое название предмета |
| `full_spec` | string | Полное название предмета |
| `teacher` | string | Имя преподавателя |

**Тип ответа:** Массив объектов
