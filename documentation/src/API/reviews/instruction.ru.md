# GET /reviews/index/instruction

Получить инструкции по отправке отзывов.

## Назначение

Отображает инструкции по написанию отзывов ученику перед отправкой обратной связи, объясняя, как следует писать отзывы.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| — | string \| null | Текст инструкции по отзывам или `null`, если инструкция недоступна |

**Тип ответа:** Одиночный объект (строка или null)
