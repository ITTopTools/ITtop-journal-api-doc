# Отзывы

Отзывы об уроках и система обратной связи учеников — просмотр отзывов преподавателей, отправка оценок и отслеживание публикаций отзывов на внешних площадках.

## Эндпоинты

| Method | Path | Краткое описание | Ссылка |
|--------|------|------------------|--------|
| GET | `/reviews/index/list` | Список отзывов об уроках | [Подробнее](list.md) |
| GET | `/reviews/index/instruction` | Получить инструкции по отзывам | [Подробнее](instruction.md) |
| GET | `/feedback/students/evaluate-lesson-list` | Список уроков для оценки | [Подробнее](evaluate-lesson.md) |
| GET | `/feedback/social-review/get-review-list` | Список отправленных социальных отзывов | [Подробнее](social-review.md) |
