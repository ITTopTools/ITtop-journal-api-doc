# GET /feedback/social-review/get-review-list

Получить список отправленных социальных отзывов ученика.

## Назначение

Отслеживание социальных отзывов — показывает, какие отзывы ученик опубликовал на внешних площадках, их статус одобрения и комментарии преподавателей.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `status` | string | Статус отправки отзыва |
| `social_id` | integer | Идентификатор платформы социального отзыва |
| `link_id` | integer | Идентификатор ссылки |
| `link` | string | URL опубликованного отзыва |
| `screen_shot` | string | URL скриншота отзыва |
| `review_id` | integer | Уникальный идентификатор отзыва |
| `comment` | string | Комментарий преподавателя к отзыву |
| `teach_name` | string | Имя преподавателя |
| `updated_at` | string | Время последнего обновления |
| `is_visibility` | boolean | Видимость отзыва |

**Тип ответа:** Массив объектов
