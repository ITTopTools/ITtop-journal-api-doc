# Главная панель

Обзорные данные, отображаемые на главной странице ученика: графики прогресса, посещаемость, рейтинги и предстоящие экзамены.

## Конечные точки

### GET /dashboard/chart/average-progress

Ежемесячные средние значения прогресса ученика.

**Назначение:** Отображает линейный график прогресса на главной панели. `points` — значение текущего месяца, `previous_points` — предыдущего месяца.

**Ответ:** Массив объектов с полями `date`, `points`, `previous_points`, `has_rasp`.

### GET /dashboard/chart/attendance

Ежемесячные проценты посещаемости для группы ученика.

**Назначение:** Отображает график посещаемости на главной панели. `points` — процент посещаемости за указанный месяц.

**Ответ:** Массив объектов с полями `date`, `points`, `previous_points`, `has_rasp`.

### GET /dashboard/chart/progress

Данные графика прогресса, сгруппированные по типу графика.

**Назначение:** Мультиграфовое представление на главной панели — каждая группа представляет отдельный тип метрики. `chart_models` имеет ту же структуру, что и элементы average-progress.

**Ответ:** Массив объектов с полями `chart_type` (int), `chart_models` (массив элементов date/points).

### GET /dashboard/progress/activity

Лента активности с баллами, достижениями и значками.

**Назначение:** Отображает последние действия в разделе активности на главной панели — полученные баллы, разблокированные достижения.

**Ответ:** Массив объектов с полями `date`, `action`, `current_point`, `point_types_id`, `point_types_name`, `achievements_id`, `achievements_name`, `achievements_type`, `badge`, `old_competition`.

### GET /dashboard/progress/leader-group

Лучшие ученики в группе ученика, ранжированные по баллам.

**Назначение:** Рейтинг группы на главной панели. Каждая запись содержит позицию, имя, фото и общее количество баллов.

**Ответ:** Массив объектов с полями `id`, `full_name`, `photo_path`, `position`, `amount`.

### GET /dashboard/progress/leader-stream

Лучшие ученики всего потока, ранжированные по баллам.

**Назначение:** Рейтинг всего потока. Та же структура, что и у рейтинга группы, но охватывает все группы в потоке.

**Ответ:** Массив объектов с полями `id`, `full_name`, `photo_path`, `position`, `amount`.

### GET /dashboard/progress/leader-group-points

Позиция и статистика баллов текущего ученика в рамках группы.

**Назначение:** Карточка ранга ученика на главной панели — показывает позицию в группе, общее количество участников и изменение баллов за неделю/месяц.

**Ответ:** Объект с полями `totalCount` (размер группы), `studentPosition`, `weekDiff`, `monthDiff`.

### GET /dashboard/progress/leader-stream-points

Позиция и статистика баллов текущего ученика в рамках потока.

**Назначение:** То же, что и leader-group-points, но для всего потока. `totalCount` может быть null.

**Ответ:** Объект с полями `totalCount`, `studentPosition`, `weekDiff`, `monthDiff`.

### GET /dashboard/info/future-exams

Список предстоящих экзаменов.

**Назначение:** Уведомление о предстоящих экзаменах на главной панели.

**Ответ:** Массив объектов с полями `spec` (название предмета), `date`.
