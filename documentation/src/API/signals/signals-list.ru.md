# GET /signal/operations/signals-list

Получить список академических сигналов и оповещений, назначенных ученику.

## Назначение

Лента сигналов — отображает активные оповещения, запросы и проблемы, отмеченные академией для ученика. Используется для отображения панели раннего предупреждения.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор сигнала |
| `date_desired` | string | Желаемая дата решения |
| `date_start` | string | Дата создания сигнала |
| `priority` | string | Уровень приоритета сигнала |
| `status` | string | Текущий статус сигнала |
| `message` | string | Текст сообщения сигнала |
| `initiator_name` | string | Имя инициатора сигнала |
| `problem_id` | integer | Идентификатор связанного типа проблемы |
| `problem_name` | string | Название связанного типа проблемы |
| `problem_type` | string | Категория проблемы |
| `days_diff` | integer | Количество дней с момента создания сигнала |
| `theme` | string | Тема сигнала |

**Тип ответа:** Массив объектов
