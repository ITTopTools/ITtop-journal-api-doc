# GET /signal/operations/problems-list

Получить справочный список типов проблем, используемых в сигналах.

## Назначение

Каталог типов проблем — сопоставляет идентификаторы проблем с понятными названиями. Используется для отображения меток типов проблем рядом с данными сигналов.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор типа проблемы |
| `title` | string | Понятное название типа проблемы |

**Тип ответа:** Массив объектов
