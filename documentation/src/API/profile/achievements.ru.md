# GET /profile/statistic/student-achievements

Получить значки достижений ученика и разбивку их баллов.

## Назначение

Используется страницей профиля ученика для отображения полученных значков и связанных с ними значений баллов.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор достижения |
| `translate_key` | string | Ключ локализации для названия достижения |
| `is_active` | boolean | Активно ли достижение в данный момент |
| `achieve_points` | array | Список записей баллов для этого достижения |

Каждый элемент в `achieve_points`:

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор записи баллов |
| `points_count` | integer | Количество начисленных баллов |

**Тип ответа:** Массив объектов
