# Профиль

Раздел профиля ученика — идентификационная информация, редактируемые настройки и значки достижений.

## Эндпоинты

| Method | Path | Краткое описание | Ссылка |
|--------|------|------------------|--------|
| GET | `/settings/user-info` | Получить идентификационные данные ученика и информацию о группе | [Подробнее](user-info.md) |
| GET | `/profile/operations/settings` | Получить редактируемые настройки профиля | [Подробнее](settings.md) |
| GET | `/profile/statistic/student-achievements` | Получить значки достижений ученика | [Подробнее](achievements.md) |

## Зависимости

- [Аутентификация](../auth/index.md) — все эндпоинты требуют действительный JWT токен
