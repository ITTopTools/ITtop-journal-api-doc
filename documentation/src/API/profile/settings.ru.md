# GET /profile/operations/settings

Получить редактируемые настройки профиля ученика.

## Назначение

Используется формой редактирования профиля для отображения и изменения персональной информации ученика, такой как адрес, email и фото.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор записи профиля |
| `ful_name` | string | Полное имя (опечатка API — обратите внимание на отсутствующую `l`) |
| `address` | string | Адрес ученика |
| `date_birth` | string | Дата рождения |
| `study` | string | Информация об обучении |
| `email` | string | Email ученика |
| `last_approving_status` | string | Статус последнего согласования |
| `form_type` | string | Тип формы |
| `photo_path` | string | Путь к фото ученика |
| `has_not_approved_data` | boolean | Есть ли несогласованные данные |
| `has_not_approved_photo` | boolean | Есть ли несогласованное фото |
| `is_email_verified` | boolean | Подтверждён ли email |
| `is_phone_verified` | boolean | Подтверждён ли телефон |
| `fill_percentage` | number | Процент заполнения профиля |

**Тип ответа:** Одиночный объект
