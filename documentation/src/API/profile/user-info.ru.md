# GET /settings/user-info

Получить основные идентификационные данные ученика и информацию о группе.

## Назначение

Используется страницей профиля ученика для отображения имени, группы, потока, уровня и других идентификационных данных.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `student_id` | integer | Идентификатор ученика |
| `full_name` | string | Полное имя ученика |
| `group_name` | string | Название текущей группы |
| `stream_name` | string | Название потока |
| `level` | string | Текущий уровень обучения |
| `age` | integer | Возраст ученика |
| `gender` | string | Пол ученика |
| `birthday` | string | Дата рождения |
| `last_date_visit` | string | Дата последнего посещения |
| `registration_date` | string | Дата регистрации |
| `current_group_id` | integer | ID текущей группы — используется как `group_id` в эндпоинтах домашних заданий |
| `current_group_status` | string | Статус текущей группы |
| `stream_id` | integer | Идентификатор потока |
| `achieves_count` | integer | Количество достижений |
| `photo` | string | URL или путь к фото ученика |
| `study_form_short_name` | string | Краткое название формы обучения |
| `groups` | array | Список групп, в которых состоит ученик |

**Тип ответа:** Одиночный объект
