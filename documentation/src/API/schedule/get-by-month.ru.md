# GET /schedule/operations/get-month

Получить расписание занятий на месяц, содержащий указанную дату.

## Назначение

Обзор расписания на месяц — отображает все занятия за весь месяц, в который входит указанная дата.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `date_filter` | query | string | да | Любая дата в формате `YYYY-MM-DD` в пределах целевого месяца |

## Ответ

Возвращает общую структуру [ScheduleItem](index.md#scheduleitem-structure).

**Тип ответа:** Массив объектов ScheduleItem
