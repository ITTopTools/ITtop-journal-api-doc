# GET /schedule/operations/get-by-date-range

Получить расписание занятий на диапазон дат.

## Назначение

Просмотр расписания на неделю или месяц — возвращает занятия между начальной и конечной датами включительно. Пустой массив является нормальным результатом, если на указанный диапазон нет занятий.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `date_start` | query | string | да | Начальная дата в формате `YYYY-MM-DD` |
| `date_end` | query | string | да | Конечная дата в формате `YYYY-MM-DD` |

## Ответ

Возвращает общую структуру [ScheduleItem](index.md#scheduleitem-structure).

**Тип ответа:** Массив объектов ScheduleItem (пустой массив — нормальный результат)
