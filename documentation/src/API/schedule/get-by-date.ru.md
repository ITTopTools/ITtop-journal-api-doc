# GET /schedule/operations/get-by-date

Получить расписание занятий на указанную дату.

## Назначение

Просмотр расписания на день — отображает занятия с указанием времени, преподавателя, предмета и аудитории для одного дня.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `date_filter` | query | string | да | Дата в формате `YYYY-MM-DD` |

## Ответ

Возвращает общую структуру [ScheduleItem](index.md#scheduleitem-structure).

**Тип ответа:** Массив объектов ScheduleItem
