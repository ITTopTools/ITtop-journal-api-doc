# GET /homework/settings/group-history

Получить группы ученика с их предметами для переключателя групп.

## Назначение

Переключатель групп во вкладке домашних заданий — показывает все группы, в которых состоит ученик, с предметами (специализациями) для каждой группы. Используется для переключения между группами в интерфейсе.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID группы |
| `name` | string | Название группы |
| `specs` | array | Вложенный массив объектов GroupSpecItem (см. ниже) |

**GroupSpecItem (вложено в `specs`):**

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID предмета (специализации) |
| `name` | string | Полное название предмета |
| `short_name` | string | Сокращённое название предмета |

**Тип ответа:** Массив объектов

**Примечание:** Этот эндпоинт НЕ используется для определения `group_id`. Параметр `group_id` для запросов домашнего задания берётся из `current_group_id` в `/settings/user-info`.
