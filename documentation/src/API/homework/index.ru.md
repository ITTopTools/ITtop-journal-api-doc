# Домашнее задание

Домашние задания, счётчики статусов, теги самооценки и настройки, связанные с группой, для вкладки домашнего задания.

## Эндпоинты

| Method | Path | Краткое описание | Ссылка |
|--------|------|------------------|--------|
| GET | `/count/homework` | Счётчики статусов домашних заданий | [Подробнее](counters.md) |
| GET | `/homework/settings/group-history` | Группы ученика с предметами | [Подробнее](group-history.md) |
| GET | `/settings/group-specs` | Предметы текущей группы | [Подробнее](group-specs.md) |
| GET | `/homework/evaluation/operations/get-tags` | Теги самооценки | [Подробнее](evaluation-tags.md) |
| GET | `/homework/operations/list` | Постраничный список домашних заданий | [Подробнее](list.md) |

## Зависимости

Поток данных домашнего задания зависит от эндпоинтов из других разделов. Связь:

```
/settings/user-info
  |
  |-- current_group_id ──► параметр group_id для /homework/operations/list
  |
/settings/group-specs
  |
  |-- список предметов ──► фильтр/селектор в UI домашнего задания
  |
/count/homework
  |
  |-- счётчики статусов ──► числа на значках вкладок домашнего задания
```

1. **`/settings/user-info`** (раздел профиля) предоставляет `current_group_id`, который необходимо передать как параметр `group_id` при вызове `/homework/operations/list`.
2. **`/settings/group-specs`** (этот раздел) возвращает предметы, доступные в текущей группе ученика, используемые для фильтра по предмету в UI домашнего задания.
3. **`/count/homework`** (этот раздел) возвращает счётчики статусов, которые определяют числа на значках заголовков вкладок домашнего задания.
