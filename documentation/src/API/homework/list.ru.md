# GET /homework/operations/list

Получить постраничный список домашних заданий, отфильтрованных по статусу, типу и группе.

## Назначение

Основной список домашних заданий во вкладке ДЗ — возвращает задания, отфильтрованные по статусу, типу и группе. Это основной эндпоинт, обеспечивающий UI страницы домашних заданий.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `page` | query | integer | да | Номер страницы для пагинации |
| `status` | query | integer | нет | Фильтр по статусу: 0=просрочено, 1=проверено, 2=сдано, 3=текущее, 5=удалено |
| `type` | query | integer | нет | 0=обычное домашнее задание, 1=лабораторная работа (идентичная структура ответа) |
| `group_id` | query | integer | да | Из `current_group_id` в `/settings/user-info` |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID домашнего задания |
| `id_spec` | integer | ID предмета (специализации) |
| `id_teach` | integer | ID преподавателя |
| `id_group` | integer | ID группы |
| `fio_teach` | string | ФИО преподавателя |
| `theme` | string | Тема/заголовок задания |
| `completion_time` | string \| null | Срок сдачи |
| `creation_time` | string | Дата создания задания |
| `overdue_time` | string \| null | Дата просрочки задания |
| `filename` | string \| null | Имя исходного прикреплённого файла |
| `file_path` | string \| null | Путь к прикреплённому файлу |
| `comment` | string \| null | Описание/комментарий к заданию |
| `name_spec` | string | Название предмета |
| `status` | integer | Статус задания (соответствует значениям counter_type) |
| `common_status` | integer | Агрегированный статус для отображения |
| `cover_image` | string \| null | URL или путь к обложке |
| `homework_stud` | object \| null | Вложенная сдача ученика (см. ниже) |
| `homework_comment` | object \| null | Вложенный комментарий преподавателя (см. ниже) |

**homework_stud (вложенная сдача ученика):**

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID сдачи |
| `filename` | string \| null | Имя сданного файла |
| `file_path` | string \| null | Путь к сданному файлу |
| `tmp_file` | string \| null | Ссылка на временный файл |
| `mark` | string \| null | Оценка, выставленная преподавателем |
| `creation_time` | string | Дата сдачи учеником |
| `stud_answer` | string \| null | Текстовый ответ ученика |
| `auto_mark` | string \| null | Автоматическая оценка (например, при просрочке) |

**homework_comment (вложенный комментарий преподавателя):**

| Поле | Тип | Описание |
|------|-----|----------|
| `text_comment` | string \| null | Текст комментария преподавателя |
| `attachment` | string \| null | Имя прикреплённого файла комментария |
| `attachment_path` | string \| null | Путь к прикреплённому файлу комментария |
| `date_updated` | string \| null | Дата последнего обновления комментария |

**Тип ответа:** Массив объектов HomeworkItem
