# GET /count/homework

Получить счётчики статусов домашних заданий.

## Назначение

Значки на вкладках страницы домашних заданий — показывает количество элементов в каждой категории статуса, позволяя фронтенду отображать числа на значках вкладок.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `counter_type` | integer | Идентификатор категории статуса (см. таблицу ниже) |
| `counter` | integer | Количество элементов в этой категории |

**Значения counter_type:**

| Значение | Значение |
|----------|----------|
| 0 | Просрочено — срок сдачи прошёл, автоматически выставлена оценка «неуд» |
| 1 | Проверено преподавателем — имеется оценка |
| 2 | Сдано — ожидает проверки преподавателем |
| 3 | Текущее — ещё не сдано учеником |
| 4 | Общее количество по всем типам |
| 5 | Удалено преподавателем — было сдано, но отклонено |

**Тип ответа:** Массив объектов
