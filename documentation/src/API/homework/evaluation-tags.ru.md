# GET /homework/evaluation/operations/get-tags

Получить теги оценки домашнего задания.

## Назначение

Теги оценки ДЗ, отображаемые ученику после сдачи домашнего задания — ученик выбирает теги, описывающие его уверенность и уровень усилий.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID тега |
| `translate_key` | string | Ключ перевода для рендеринга i18n |
| `type` | integer | Тип/категория тега |

**Тип ответа:** Массив объектов
