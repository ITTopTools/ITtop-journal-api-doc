# GET /settings/group-specs

Получить предметы (специализации), привязанные к текущей группе ученика.

## Назначение

Фильтр/селектор предметов в UI домашних заданий — показывает, какие предметы изучает текущая группа, позволяя ученику фильтровать домашние задания по предмету.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | ID предмета (специализации) |
| `name` | string | Полное название предмета |
| `short_name` | string | Сокращённое название предмета |

**Тип ответа:** Массив объектов
