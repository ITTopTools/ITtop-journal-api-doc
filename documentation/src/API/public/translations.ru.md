# GET /public/translations

Получить пары ключ-значение локализации для заданного языка.

## Назначение

Строки переводов интерфейса — возвращает плоский словарь ключей переводов, сопоставленных с локализованными строками. Используется фронтендом для отображения всего текста на выбранном языке.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `language` | query | string | да | Код локали (например, "ru") |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| *(динамические ключи)* | string | Ключ перевода, сопоставленный с локализованной строкой |

**Тип ответа:** Одиночный объект (плоский словарь — тысячи ключей)
