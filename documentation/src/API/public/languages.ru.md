# GET /public/languages

Получить список доступных языков интерфейса приложения.

## Назначение

Справочник для выбора языка — список поддерживаемых приложением локалей. Используется для заполнения переключателя языка в интерфейсе.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `name_mystat` | string | Отображаемое название языка |
| `short_name` | string | Код локали (например, "ru") |

**Тип ответа:** Массив объектов
