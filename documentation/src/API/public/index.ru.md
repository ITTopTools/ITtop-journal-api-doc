# Публичные данные

Публичные справочные данные — аутентификация не требуется. Эти эндпоинты можно вызывать без Bearer токена.

## Эндпоинты

| Method | Path | Краткое описание | Ссылка |
|--------|------|------------------|--------|
| GET | `/public/languages` | Список доступных языков | [Подробнее](languages.md) |
| GET | `/public/translations` | Получить строки переводов | [Подробнее](translations.md) |
| GET | `/public/tags` | Список публичных тегов | [Подробнее](tags.md) |
