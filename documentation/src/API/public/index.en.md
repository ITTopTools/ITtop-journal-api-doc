# Public

Public reference data — no authentication required. These endpoints can be called without a Bearer token.

## Endpoints

| Method | Path | Summary | Link |
|--------|------|---------|------|
| GET | `/public/languages` | List available languages | [Details](languages.md) |
| GET | `/public/translations` | Get translation strings | [Details](translations.md) |
| GET | `/public/tags` | List public tags | [Details](tags.md) |
