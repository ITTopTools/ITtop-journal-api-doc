# Schedule

## Endpoints

| Method | Path | Description | Parameters |
|--------|------|-------------|------------|
| GET | `/schedule/operations/get-by-date` | Schedule for a date | `?date_filter=YYYY-MM-DD` |
| GET | `/schedule/operations/get-by-date-range` | Schedule for a range | `?date_start&date_end=YYYY-MM-DD` |
| GET | `/schedule/operations/get-month` | Schedule for a month | `?date_filter=YYYY-MM-DD` |
