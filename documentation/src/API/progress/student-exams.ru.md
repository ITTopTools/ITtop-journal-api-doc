# GET /progress/operations/student-exams

Получить результаты экзаменов с оценками и прикреплёнными файлами для аутентифицированного ученика.

## Назначение

Страница оценок за экзамены — отображает баллы за экзамены, комментарии преподавателя и прикреплённые файлы для каждого сданного учеником экзамена.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `teacher` | string | Преподаватель, проводивший экзамен |
| `mark` | string \| null | Оценка/балл за экзамен |
| `mark_type` | integer | Тип оценки (см. примечание ниже) |
| `date` | string | Дата экзамена |
| `ex_file_name` | string \| null | Имя файла экзамена |
| `id_file` | integer \| null | ID файла |
| `exam_id` | integer | Уникальный идентификатор экзамена |
| `file_path` | string \| null | Путь к прикреплённому файлу |
| `comment_teach` | string \| null | Комментарий преподавателя к экзамену |
| `need_access` | boolean | Требуется ли запрос доступа к файлу |
| `spec` | string | Название предмета/специализации |

**Тип ответа:** Массив объектов
