# Прогресс

Отслеживание учебного прогресса: записи посещений с оценками за занятие и результаты экзаменов с прикреплёнными файлами.

## Эндпоинты

| Method | Path | Краткое описание | Ссылка |
|--------|------|------------------|--------|
| GET | `/progress/operations/student-visits` | Записи посещений с оценками за занятие | [Подробнее](student-visits.md) |
| GET | `/progress/operations/student-exams` | Результаты экзаменов с оценками и файлами | [Подробнее](student-exams.md) |

## Зависимости

Оба эндпоинта требуют действительный JWT токен из раздела [Аутентификация](../auth/index.md).
