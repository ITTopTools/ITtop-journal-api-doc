# GET /progress/operations/student-visits

Получить записи посещений с оценками за занятие для аутентифицированного ученика.

## Назначение

Страница журнала посещений — отображает посещаемость и оценки за каждое занятие. Каждая запись включает статус посещения и отдельные поля оценок для разных видов работ (контрольная, домашняя работа, лабораторная, классная работа, практическая работа, итоговая работа).

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| *(нет)* | | | | Этот эндпоинт не принимает параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `date_visit` | string | Дата посещения (`YYYY-MM-DD`) |
| `lesson_number` | integer | Номер занятия в течение дня |
| `status_was` | string | Статус посещения |
| `spec_id` | integer | ID предмета (специализации) |
| `teacher_name` | string | ФИО преподавателя |
| `spec_name` | string | Название предмета |
| `lesson_theme` | string | Тема занятия |
| `control_work_mark` | string \| null | Оценка за контрольную работу |
| `home_work_mark` | string \| null | Оценка за домашнюю работу |
| `lab_work_mark` | string \| null | Оценка за лабораторную работу |
| `class_work_mark` | string \| null | Оценка за классную работу |
| `practical_work_mark` | string \| null | Оценка за практическую работу |
| `final_work_mark` | string \| null | Оценка за итоговую работу |

**Тип ответа:** Массив объектов
