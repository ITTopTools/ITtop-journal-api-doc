# GET /library/operations/list

Получить список учебных материалов из библиотеки.

## Назначение

Используется разделом библиотеки в приложении для отображения доступных учебных материалов, отфильтрованных по типу и категории.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `material_type` | query | int | нет | Фильтр по типу материала (по умолчанию: 2) |
| `filter_type` | query | int | нет | Тип фильтра (по умолчанию: 0) |
| `recommended_type` | query | int | нет | Тип рекомендации (по умолчанию: 0) |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| — | — | Схема ответа неизвестна — собранные данные были пустыми |

**Тип ответа:** Массив объектов
