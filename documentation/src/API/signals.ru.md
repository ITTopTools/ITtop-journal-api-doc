# Сигналы

Учебные сигналы оповещения и индикаторы проблем — система раннего предупреждения об учебных проблемах.

## Эндпоинты

### GET /signal/operations/signals-list

Учебные сигналы/оповещения, назначенные студенту.

**Назначение:** Лента сигналов — отображает активные оповещения, запросы и проблемы, которые академия отметила для студента.

**Ответ:** Массив объектов с полями `id`, `date_desired`, `date_start`, `priority`, `status`, `message`, `initiator_name`, `problem_id`, `problem_name`, `problem_type`, `days_diff`, `theme`.

### GET /signal/operations/problems-list

Справочный список типов проблем, используемых в сигналах.

**Назначение:** Каталог типов проблем — сопоставляет идентификаторы проблем с читаемыми названиями.

**Ответ:** Массив объектов с полями `id`, `title`.
