# Отзывы

Отзывы о занятиях и обратная связь от студентов.

## Эндпоинты

### GET /reviews/index/list

Отзывы, написанные о занятиях.

**Назначение:** Отображает отзывы преподавателей в профиле студента. Каждая запись включает дату отзыва, текст сообщения, предмет и имя преподавателя.

**Ответ:** Массив объектов с полями `date`, `message`, `spec` (краткое название предмета), `full_spec` (полное название предмета), `teacher`.

### GET /reviews/index/instruction

Инструкции для отправки отзывов.

**Назначение:** Руководство по отправке отзывов, отображаемое студенту.

**Ответ:** Текст инструкции по отзывам или `null`, если инструкция недоступна.

### GET /feedback/students/evaluate-lesson-list

Занятия, которые студент может оценить (оставить обратную связь).

**Назначение:** Форма обратной связи — отображает прошедшие занятия, на которых студент может оценить преподавателя. Каждая запись содержит дату занятия, имя преподавателя и предмет.

**Ответ:** Массив объектов с полями `key`, `date_visit`, `fio_teach`, `spec_name`, `teach_photo`.

### GET /feedback/social-review/get-review-list

Отправленные студентом социальные отзывы (например, отзывы о курсах на внешних платформах).

**Назначение:** Отслеживание социальных отзывов — отображает, какие отзывы отправил студент, их статус одобрения и комментарии преподавателя.

**Ответ:** Массив объектов с полями `status`, `social_id`, `link_id`, `link`, `screen_shot`, `review_id`, `comment`, `teach_name`, `updated_at`, `is_visibility`.
