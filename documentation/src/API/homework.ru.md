# Домашнее задание

Домашние задания, счётчики, теги самооценки и настройки, связанные с группой, для вкладки ДЗ (домашнее задание).

## Конечные точки

### GET /count/homework

Количество домашних заданий по типам статуса.

**Назначение:** Значки на вкладке домашнего задания — показывает, сколько элементов в каждой категории статуса.

**Ответ:** Массив объектов с полями `counter_type`, `counter`.

**Значения counter_type:**

| Значение | Значение |
|----------|----------|
| 0 | Просрочено — срок сдачи прошёл, автоматически выставлена оценка «неуд» |
| 1 | Проверено преподавателем — имеется оценка |
| 2 | Сдано — ожидает проверки преподавателем |
| 3 | Текущее — ещё не сдано учеником |
| 4 | Общее количество по всем типам |
| 5 | Удалено преподавателем — было сдано, но отклонено |

### GET /homework/settings/group-history

Группы ученика с их предметами (специализациями).

**Назначение:** Переключатель групп во вкладке ДЗ. Показывает все группы, в которых состоит ученик, с предметами для каждой.

**Ответ:** Массив объектов с полями `id`, `name`, `specs` (массив `GroupSpecItem`).

Примечание: `group_id` для запросов домашнего задания берётся из `current_group_id` в `/settings/user-info`, а не из этой конечной точки.

### GET /settings/group-specs

Предметы (специализации), привязанные к текущей группе ученика.

**Назначение:** Фильтр/селектор предметов для домашнего задания — показывает, какие предметы изучает текущая группа.

**Ответ:** Массив объектов с полями `id`, `name`, `short_name`.

### GET /homework/evaluation/operations/get-tags

Теги для формы самооценки домашнего задания.

**Назначение:** Теги самооценки, отображаемые ученику после сдачи домашнего задания.

**Ответ:** Массив объектов с полями `id`, `translate_key`, `type`.

### GET /homework/operations/list

Постраничный список домашних заданий.

**Назначение:** Основной список домашних заданий во вкладке ДЗ. Возвращает задания, отфильтрованные по статусу, типу и группе.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `page` | int | Номер страницы для пагинации |
| `status` | int | Фильтр: 0=просрочено, 1=проверено, 2=сдано, 3=текущее, 5=удалено |
| `type` | int | 0=обычное домашнее задание, 1=лабораторная работа (идентичная структура ответа) |
| `group_id` | int | Из `current_group_id` в `/settings/user-info` |

**Ответ:** Массив объектов `HomeworkItem` с полями `id`, `id_spec`, `id_teach`, `id_group`, `fio_teach`, `theme`, `completion_time`, `creation_time`, `overdue_time`, `filename`, `file_path`, `comment`, `name_spec`, `status`, `common_status`, `cover_image`, `homework_stud` (вложенная сдача), `homework_comment` (вложенный комментарий преподавателя).
