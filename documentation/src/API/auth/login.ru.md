# POST /auth/login

Отправить учётные данные для получения JWT токенов для доступа к API.

## Назначение

Используется формой входа на фронтенде для аутентификации ученика и получения токенов, необходимых для всех последующих вызовов API.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| `application_key` | body | string | да | Ключ приложения (фиксированное значение) |
| `id_city` | body | number \| null | нет | ID города |
| `username` | body | string | да | Логин ученика |
| `password` | body | string | да | Пароль ученика |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `access_token` | string | JWT access token для авторизации в API |
| `refresh_token` | string | JWT refresh token |
| `expires_in_access` | integer | Время истечения access token в секундах |
| `expires_in_refresh` | integer | Время истечения refresh token в секундах |
| `user_type` | string | Тип аутентифицированного пользователя |
| `city_data` | object | Информация о городе пользователя |
| `user_role` | string | Роль, назначенная пользователю |

**Тип ответа:** Одиночный объект
