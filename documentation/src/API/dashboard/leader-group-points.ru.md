# GET /dashboard/progress/leader-group-points

Получить текущую позицию ученика в таблице лидеров группы.

## Назначение

Используется панелью управления для отображения ранга ученика, общего числа участников и изменений позиции за неделю/месяц внутри группы.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `totalCount` | integer | Общее количество учеников в группе |
| `studentPosition` | integer | Текущая позиция ученика в группе |
| `weekDiff` | integer | Изменение позиции по сравнению с прошлой неделей |
| `monthDiff` | integer | Изменение позиции по сравнению с прошлым месяцем |

**Тип ответа:** Одиночный объект (не список)
