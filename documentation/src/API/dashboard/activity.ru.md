# GET /dashboard/progress/activity

Получить ленту активности ученика.

## Назначение

Используется панелью управления для отображения хронологического списка действий ученика, изменений баллов и разблокированных достижений.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `date` | string | Дата активности |
| `action` | string | Описание выполненного действия |
| `current_point` | number | Баланс баллов после действия |
| `point_types_id` | integer | Идентификатор типа баллов |
| `point_types_name` | string | Название типа баллов |
| `achievements_id` | integer | Идентификатор связанного достижения |
| `achievements_name` | string | Название связанного достижения |
| `achievements_type` | string | Тип связанного достижения |
| `badge` | string | Идентификатор значка или путь |
| `old_competition` | boolean | Относится ли это к старому соревнованию |

**Тип ответа:** Массив объектов
