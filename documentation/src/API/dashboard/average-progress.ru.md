# GET /dashboard/chart/average-progress

Получить данные графика среднемесячного прогресса.

## Назначение

Используется панелью управления для отображения графика тренда среднего прогресса с сравнением текущих и предыдущих оценок за период.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `date` | string | Дата точки данных (формат YYYY-MM-DD) |
| `points` | number | Оценка текущего периода |
| `previous_points` | number | Оценка предыдущего периода |
| `has_rasp` | boolean | Относится ли данных к записи расписания |

**Тип ответа:** Массив объектов
