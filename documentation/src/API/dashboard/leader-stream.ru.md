# GET /dashboard/progress/leader-stream

Получить рейтинг таблицы лидеров потока.

## Назначение

Используется панелью управления для отображения ранжированного списка учеников всего потока на основе их суммы баллов.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Идентификатор ученика |
| `full_name` | string | Полное имя ученика |
| `photo_path` | string | Путь к фото ученика |
| `position` | integer | Позиция в рейтинге потока |
| `amount` | number | Итого баллов |

**Тип ответа:** Массив объектов
