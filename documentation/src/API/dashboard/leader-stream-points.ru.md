# GET /dashboard/progress/leader-stream-points

Получить текущую позицию ученика в таблице лидеров потока.

## Назначение

Используется панелью управления для отображения ранга ученика, общего числа участников и изменений позиции за неделю/месяц по всему потоку.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `totalCount` | integer \| null | Общее количество учеников в потоке (может быть null) |
| `studentPosition` | integer | Текущая позиция ученика в потоке |
| `weekDiff` | integer | Изменение позиции по сравнению с прошлой неделей |
| `monthDiff` | integer | Изменение позиции по сравнению с прошлым месяцем |

**Тип ответа:** Одиночный объект (не список)
