# GET /dashboard/info/future-exams

Получить список предстоящих экзаменов.

## Назначение

Используется панелью управления для отображения запланированных будущих экзаменов ученика, включая предмет и дату.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `spec` | string | Предмет или специальность экзамена |
| `date` | string | Дата экзамена |

**Тип ответа:** Массив объектов
