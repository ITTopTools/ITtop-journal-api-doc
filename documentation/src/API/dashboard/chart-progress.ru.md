# GET /dashboard/chart/progress

Получить данные прогресса, сгруппированные по типу графика.

## Назначение

Используется панелью управления для отображения нескольких графиков прогресса, каждый из которых идентифицируется значением типа графика и содержит собственный набор моделей данных.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `chart_type` | integer | Идентификатор типа графика |
| `chart_models` | array | Массив моделей данных графика для этого типа |

Каждый элемент в `chart_models` имеет ту же структуру, что и графические эндпоинты (см. [Средний прогресс](average-progress.md)).

**Тип ответа:** Массив объектов
