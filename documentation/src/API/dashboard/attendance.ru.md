# GET /dashboard/chart/attendance

Получить данные графика ежемесячного процента посещаемости.

## Назначение

Используется панелью управления для отображения графика тренда посещаемости с сравнением текущих и предыдущих показателей за период.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Нет параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `date` | string | Дата точки данных (формат YYYY-MM-DD) |
| `points` | number | Процент посещаемости текущего периода |
| `previous_points` | number | Процент посещаемости предыдущего периода |
| `has_rasp` | boolean | Относится ли данных к записи расписания |

**Тип ответа:** Массив объектов
