# GET /news/operations/latest-news

Получить последние новости академии.

## Назначение

Лента новостей на панели управления — отображает объявления, события и обновления академии. Включает отслеживание прочитанного/непрочитанного для каждого ученика.

## Параметры

| Параметр | In | Тип | Обязательный | Описание |
|----------|----|-----|--------------|----------|
| — | — | — | — | Без параметров |

## Ответ

| Поле | Тип | Описание |
|------|-----|----------|
| `id_bbs` | integer | Идентификатор новости |
| `theme` | string | Заголовок/тема новости |
| `time` | string | Время публикации |
| `viewed` | boolean | Прочитана ли новость учеником |

**Тип ответа:** Массив объектов
