# Dashboard

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/dashboard/chart/average-progress` | Average progress by month |
| GET | `/dashboard/chart/attendance` | Attendance by month |
| GET | `/dashboard/chart/progress` | Progress charts by type |
| GET | `/dashboard/progress/leader-group` | Top students in group |
| GET | `/dashboard/progress/leader-stream` | Top students in stream |
| GET | `/dashboard/progress/leader-group-points` | Student position and points in group |
| GET | `/dashboard/progress/leader-stream-points` | Student position and points in stream |
| GET | `/dashboard/progress/activity` | Activity feed (points, achievements) |
| GET | `/dashboard/info/future-exams` | Upcoming exams |
