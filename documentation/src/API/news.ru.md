# Новости

Новости и объявления академии.

## Конечные точки

### GET /news/operations/latest-news

Последние новости академии.

**Назначение:** Лента новостей на главной панели — показывает объявления, события и обновления академии.

**Ответ:** Массив объектов с полями `id_bbs`, `theme`, `time`, `viewed` (прочитал ли ученик данную новость).
