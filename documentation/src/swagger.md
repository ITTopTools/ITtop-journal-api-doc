# Swagger UI

Interactive API documentation powered by Swagger UI.

## How to Authorize

1. Click the **Authorize** button
2. Enter your Bearer token
3. All subsequent "Try it out" requests will include the token

## Without a Token

If you don't authorize, the mock server returns anonymized example data. This is enough to explore the API structure.

## With a Token

Authorized requests go through to the real API. See [Authentication](authentication.md) for how to get a token.
