# Swagger UI

Интерактивная документация API на базе Swagger UI.

## Как авторизоваться

1. Нажмите кнопку **Authorize**
2. Введите Bearer-токен
3. Все последующие запросы «Try it out» будут включать токен

## Без токена

Без авторизации mock-сервер вернёт анонимизированные примеры данных. Этого достаточно для изучения структуры API.

## С токеном

Авторизованные запросы проксируются к реальному API. См. [Аутентификация](authentication.md) для получения токена.
