## Обработка ошибок

Как система обрабатывает ошибки на каждом уровне?

| Уровень | Стратегия |
|---------|-----------|
| Валидация входных данных | Возвращаем 400 + описание |
| Ошибка БД | Логируем, возвращаем 500 |
| Внешний сервис недоступен | Retry x3, потом fallback / ошибка |
| Неожиданное исключение | Логируем stack trace, возвращаем 500 |
